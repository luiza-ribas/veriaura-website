# VeriAura Website

A professional, conversion-focused website for VeriAura - a consulting firm specializing in human-centered AI and automation solutions for hospitality sales teams.

## Project Overview

This website is designed to generate leads and build trust with potential clients, specifically targeting Directors of Sales in the hospitality industry with event revenue between $1M-$5M.

### Core Value Proposition
"We make AI work in real life — through strategy, systems, and skills."

## Design System

### Brand Colors
- **Navy**: #0A1931 (Primary brand color)
- **Blue**: #4A90E2 (Accent color)
- **Violet**: #9013FE (Gradient accent)
- **Off-White**: #F8F9FA (Background)
- **Light Grey**: #E9ECEF (Secondary background)

### Typography
- **Font Family**: Inter (with system font fallbacks)
- **Headings**: Bold, 700 weight
- **Body**: Regular, 18px base size

### Key Features
- **Responsive Design**: Mobile-first approach with breakpoints at 768px
- **Clean Code**: Semantic HTML5, organized CSS with CSS variables
- **Accessibility**: Proper heading hierarchy, alt text for images
- **Performance**: Minimal dependencies, optimized assets

## Deployment

### Hosting on Hostinger

1. **Upload Files**:
   - Connect to your Hostinger account via FTP or File Manager
   - Upload all files maintaining the directory structure
   - Ensure `index.html` is in the root directory

2. **Domain Configuration**:
   - Point your domain to Hostinger's nameservers
   - Configure SSL certificate (usually automatic with Hostinger)

3. **Testing**:
   - Verify all pages load correctly
   - Test navigation links
   - Check responsive design on mobile devices

### Local Development

To test locally:
1. Simply open `index.html` in your web browser
2. All pages should work without a server (static HTML/CSS)
